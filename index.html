Installation Guide

Chrome / Chromium-based browsers

Works with Chrome, Edge, Opera, Brave, and other Chromium-based browsers

Option 1: Download ZIP

  1. Download the ZIP file from GitHub and extract it
  2. Open your browser and go to the extensions page:
    • Chrome: chrome://extensions
    • Edge: edge://extensions
    • Opera: opera://extensions
    • Brave: brave://extensions
  3. Enable Developer mode in the top right corner
  4. Click on Load unpacked and select the extension/ folder
  5. Go to instagram.com, log in, and visit your profile
  6. Click on the extension icon in the toolbar and run the checker

Option 2: Clone with Git

git clone https://github.com/ValBoschP/FollowBack-Checker.git

Firefox

Installation for Mozilla Firefox browser

Option 1: Download ZIP

  1. Download the ZIP file from GitHub and extract it
  2. Open Firefox and go to about:debugging
  3. Click on This Firefox in the left sidebar
  4. Click on Load Temporary Add-on
  5. Navigate to the extension/ folder and select the manifest.json file
  6. Go to instagram.com, log in, and visit your profile
  7. Click on the extension icon in the toolbar and run the checker

Option 2: Clone with Git
